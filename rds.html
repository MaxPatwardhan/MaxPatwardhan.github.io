Haptic/Robot Telemanipulation System

Argonne National Laboratory and Northwestern University

Selective Capstone Project - June 2024

This project, developed as part of Northwestern University's selective capstone program in collaboration with Argonne National Laboratory, focused on designing and implementing a sophisticated haptic-robot telemanipulation system. This system enables users to control a robotic finger and thumb using a haptic exoskeleton. I led the design, assembly, testing, and integration of the electronics for this project, creating dedicated PCB's for power, communication and control, and sensing.

❮ ❯

Project Goals

  • Develop a responsive, high-precision telemanipulation system for tasks requiring tactile feedback.
  • Design and integrate modular PCBs to support haptic feedback and accurate robotic control.
  • Enable a user interface that allows remote manipulation in potentially hazardous or sensitive environments.

Design and Technical Details

The system utilizes two custom-designed PCBs with the following features:

  • Multilayer Architecture: Each PCB layer was optimized for signal integrity and minimized latency in transmitting haptic feedback.
  • Modularity: The modular PCB design allows for easy upgrades and maintenance, promoting long-term adaptability in different experimental setups.
  • Embedded Sensors: Advanced sensors on the PCB enable real-time feedback for high-accuracy control, crucial for manipulating fine details.

Key Achievements

  • Achieved sub-millisecond response times for haptic feedback, ensuring near-instantaneous robotic control.
  • Successfully deployed the system in a controlled environment at Argonne National Laboratory, demonstrating its capability in handling delicate tasks.
  • Integrated advanced feedback algorithms, allowing smooth and precise manipulations across varied robotic tasks.

Conclusion

This haptic/robot telemanipulation system is a step forward in the integration of robotics with intuitive, real-time user interfaces. With this system, tasks previously requiring physical presence in challenging environments can now be managed remotely with precision and responsiveness.
